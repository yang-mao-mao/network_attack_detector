## 快速启动

### 启动 GUI

```bash
cd network_attack_detector
python main.py --gui
```

程序启动后将显示主窗口，包含流量监控、告警展示、规则管理和统计分析四大功能区。

### 依赖要求

确保已安装所需依赖：

```bash
pip install PyQt6>=6.6.0 matplotlib>=3.8.0 pyecharts>=2.0.0
```

## 主窗口概览

主窗口从上到下分为四个区域：

```
┌──────────────────────────────────────────────────┐
│  菜单栏 + 工具栏                                   │
├─────────────────────┬────────────────────────────┤
│                     │                            │
│  实时流量表格        │  实时告警表格                │
│  (左侧 60%)         │  (右侧 40%)                 │
│                     │                            │
├─────────────────────┴────────────────────────────┤
│  [📊 统计图表]  [📋 规则管理]  (底部 Tab 页签)     │
├──────────────────────────────────────────────────┤
│  状态栏: 抓包状态 | 数据包计数 | 告警计数 | 运行时间  │
└──────────────────────────────────────────────────┘
```

### 工具栏按钮

| 按钮 | 作用 | 说明 |
|------|------|------|
| 网卡选择下拉框 | 选择抓包网卡 | 默认"(自动选择)" |
| **▶ 开始抓包** | 启动实时抓包检测 | 点击后按钮变灰，停止按钮可用 |
| **■ 停止** | 停止抓包 | 仅在抓包运行时可用 |
| **⏸ 暂停** | 暂停/恢复抓包 | 暂停时不处理新包但不丢失检测状态 |
| **📄 导出 CSV** | 导出告警到 CSV | 弹出保存文件对话框 |

### 状态栏指示灯

| 颜色 | 状态 | 含义 |
|------|------|------|
| 🟢 绿色 | 运行中 | 正在实时抓包和检测 |
| 🟡 黄色 | 已暂停 | 抓包已暂停 |
| ⚫ 灰色 | 已停止 | 未启动抓包 |

## 实时流量表格

### 如何查看流量

1. 启动抓包后，流量表格会自动填充捕获到的数据包
2. 每一行代表一个数据包，包含源/目的 IP、端口、协议、长度等信息
3. HTTP 协议的数据包会有**浅蓝色背景**，便于识别 Web 流量
4. 表格会自动滚动到最新数据，始终显示最新的流量

### 过滤流量

使用表格顶部的**协议下拉框**可以按协议过滤：

- 选择"全部"→ 显示所有协议的流量
- 选择"HTTP"→ 只显示 HTTP 流量
- 选择"TCP"→ 只显示 TCP 流量
- 以此类推

### 右键菜单

在流量表格的任意行上右键，可以使用以下功能：

| 菜单项 | 功能 |
|--------|------|
| **查看详情** | 弹出对话框，显示该数据包的完整信息（包括 HTTP 请求头、payload 等） |
| **复制源 IP** | 将源 IP 地址复制到剪贴板 |
| **复制目的 IP** | 将目的 IP 地址复制到剪贴板 |

### 双击操作

双击任意流量行可以快速查看该数据包的详细信息。

## 实时告警表格

### 如何查看告警

1. 检测引擎发现攻击时会自动在告警表格中新增一行
2. 告警按等级以不同颜色标记：
   - **Critical (严重)**: 红色背景，白色粗体
   - **High (高危)**: 浅橙色背景
   - **Medium (中危)**: 浅黄色背景
   - **Low (低危)**: 浅蓝色背景
3. 表格上方显示告警统计摘要：总数 + 各级别计数

### 过滤告警

使用表格顶部的下拉框可以过滤告警：

- **等级过滤**: 选择"Critical"只看严重告警
- **类型过滤**: 选择"SQL Injection"只看 SQL 注入告警
- 两个过滤条件可以组合使用

### 右键菜单

在告警表格的任意行上右键：

| 菜单项 | 功能 |
|--------|------|
| **查看详情** | 弹出对话框显示完整告警信息（证据、描述、建议） |
| **复制证据** | 将告警证据文本复制到剪贴板 |
| **过滤此 IP** | 按该源 IP 过滤告警（功能预留） |

### 处理告警的建议流程

1. 看到 Critical（红色）告警时，**立即双击**查看详情
2. 阅读 `evidence` 字段了解攻击载荷
3. 阅读 `suggestion` 字段获取处理建议
4. 根据需要导出告警数据用于报告

## 规则管理面板

规则管理面板位于底部 Tab 页签中，点击 **📋 规则管理** 标签页即可打开。

### 查看规则

面板有两个子标签页：

- **特征规则 (Signature)**: 基于字符串/正则匹配的检测规则，如 SQL 注入、XSS 检测
- **行为规则 (Behavior)**: 基于时间窗口计数的检测规则，如端口扫描、暴力破解检测

### 搜索规则

1. 在顶部搜索框中输入关键词
2. 系统会实时过滤，匹配规则 ID、名称和模式字段
3. 勾选"仅显示启用的规则"可以只查看活跃规则

### 启用/禁用规则

- **单条切换**: 点击每条规则最左侧的 ☑ checkbox，即可启用或禁用该规则
- **批量操作**: 
  1. 在表格中选中多行（Ctrl+点击 或 Shift+点击）
  2. 点击底部 **✓ 启用选中** 或 **✗ 禁用选中** 按钮

### 添加规则

1. 切换到对应规则类型的标签页（特征规则或行为规则）
2. 点击底部 **＋ 添加规则** 按钮
3. 在弹出的对话框中填写规则 ID 和名称
4. 点击"确定"创建规则（其他字段可在表格中双击编辑）

### 删除规则

1. 在表格中选中要删除的规则（可多选）
2. 点击 **－ 删除选中** 按钮
3. 在确认对话框中点击"是"

> ⚠️ 删除操作不可撤销，请谨慎操作。

## 统计面板

统计面板位于底部 Tab 页签中，点击 **📊 统计图表** 标签页即可打开。

### 图表说明

| 图表 | 位置 | 说明 |
|------|------|------|
| **饼图** — 攻击类型分布 | 左上 | 展示各类型攻击的占比，帮助了解主要威胁类型 |
| **柱状图** — Top 10 攻击源 IP | 右上 | 展示告警最多的 10 个来源 IP，快速定位攻击者 |
| **折线图** — 告警时间趋势 | 下方 | 展示最近 60 分钟的告警变化曲线（总告警/Critical/High） |

### 图表自动刷新

- 统计图表每 **5 秒** 自动刷新一次
- 当有新的告警产生时，图表数据也会实时更新
- 饼图和柱状图反映的是**累计数据**
- 折线图反映的是**最近 60 分钟**的按分钟统计

### 统计摘要

图表下方显示关键统计指标：

| 指标 | 含义 |
|------|------|
| 总告警 | 自检测开始以来的累计告警总数 |
| 总流量 | 已捕获并分析的数据包总数 |
| 检测率 | 告警数 / 流量数 × 100% |
| 最活跃攻击者 | 产生告警最多的源 IP 及其告警次数 |
| 最常见攻击 | 出现频率最高的攻击类型 |

## 导出报告

### 导出 CSV

1. 点击菜单 **文件 → 导出告警 CSV...**（快捷键 `Ctrl+E`）
2. 或点击工具栏的 **📄 导出 CSV** 按钮
3. 选择保存位置和文件名
4. CSV 文件包含：时间、等级、类型、源IP、目的IP、规则ID、证据

### 导出 HTML 报告

1. 点击菜单 **文件 → 导出 HTML 报告...**（快捷键 `Ctrl+H`）
2. 选择保存位置
3. 生成包含告警表格的静态 HTML 页面，可直接在浏览器中打开

### 导出 pyecharts 交互式图表

1. 点击菜单 **文件 → 导出图表 (pyecharts)...**（快捷键 `Ctrl+G`）
2. 选择保存位置
3. 生成包含交互式图表的 HTML 页面：
   - 鼠标悬停可查看数据详情
   - 点击图例可切换显示/隐藏数据系列
   - 支持缩放和下载图表

## 常见操作流程

### 场景一：实时监控网络攻击

```
1. 启动程序: python main.py --gui
2. 在工具栏选择网卡（或保持默认"自动选择"）
3. 点击 ▶ 开始抓包
4. 观察流量表格和告警表格的实时更新
5. 在统计面板查看攻击趋势
6. 发现严重告警时，双击查看详情
7. 点击 ■ 停止结束监控
```

### 场景二：分析离线 pcap 文件

```
1. 启动程序
2. 菜单 抓包 → 打开 pcap 文件... (Ctrl+O)
3. 选择 pcap 文件
4. 系统自动分析文件中的所有数据包
5. 在告警表格和统计面板中查看检测结果
6. 导出结果: 文件 → 导出告警 CSV...
```

### 场景三：管理检测规则

```
1. 启动程序
2. 点击底部的 📋 规则管理 标签页
3. (可选) 搜索特定规则
4. 勾选/取消勾选规则最左侧的 checkbox 来启用/禁用规则
5. 添加新规则: 点击 ＋ 添加规则
6. 删除无用规则: 选中后点击 － 删除选中
7. 菜单 规则 → 重新加载规则 (Ctrl+R) 从文件重新加载
```

### 场景四：生成攻击检测报告

```
1. 完成一段时间的监控后
2. 先用等级过滤器筛选 Critical 和 High 告警
3. 菜单 文件 → 导出图表 (pyecharts)... (Ctrl+G)
4. 保存为 charts.html
5. 菜单 文件 → 导出告警 CSV... (Ctrl+E)
6. 保存为 alerts.csv
7. 将 charts.html 和 alerts.csv 一起提交作为检测报告
```

## 注意事项

1. **数据量限制**: 流量表格最多保留 10000 行，告警表格最多保留 5000 行，超出后自动删除最旧的数据
2. **内存管理**: 长时间运行建议定期导出告警并清除告警列表（菜单 视图 → 清除告警列表）
3. **网卡权限**: Linux 下实时抓包可能需要 root 权限或配置 `cap_net_raw` 能力
4. **Python 版本**: 建议 Python 3.10+
5. **中文字体**: 统计图表的 matplotlib 中文显示需要系统安装中文字体（如 WenQuanYi Micro Hei）
